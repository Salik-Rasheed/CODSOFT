# CodSoft Python Internship Projects

This repository contains all tasks completed as part of the CodSoft Python Internship.

---

## 📁 Tasks

### 🔹 Task 1 - To-Do List
A Python-based task manager to add, update, and delete tasks.

### 🔹 Task 2 - Calculator
A simple calculator application performing basic arithmetic operations.

### 🔹 Task 3 - Rock Paper Scissors (GUI)
A modern GUI-based game built using Tkinter with:
- Interactive buttons
- Score tracking
- Best of 5 rounds
- Progress bar and animations

###🔹 Task 4 - Password Generator 🔐
A secure password generator built using Python that creates strong and random passwords.
- Customizable password length
- Mix of uppercase, lowercase, numbers, and symbols
- User-friendly interface
- Ensures strong password security

- Random module
